# MarketPulse AI 📈

MarketPulse AI is an advanced, AI-powered stock and cryptocurrency market intelligence platform. It is designed to monitor markets, process real-time financial news, and generate transparent, evidence-based market signals using Natural Language Processing (NLP) and Technical Analysis.

## 🌟 Features

- **Multi-Asset Support**: Track both Stocks and Cryptocurrencies.
- **AI-Powered News Sentiment**: Automated 10-stage NLP pipeline for news aggregation, entity extraction, sentiment analysis, and market impact scoring.
- **Transparent Signals**: Bullish, Bearish, or Neutral signals explained with exact confidence scores, risk metrics, and factored components.
- **Dynamic User Views**:
  - *Simple Mode*: Easy-to-read summaries and trend indicators for casual investors.
  - *Advanced Mode*: In-depth fundamentals, technical indicators (RSI, MACD, Bollinger Bands), and on-chain metrics for power users.
- **Backtesting Engine**: Walk-forward backtesting system to test trading strategies with slippage and PnL metrics.
- **Real-Time Data**: WebSocket integration for live price updates.

## 🏗 Architecture

MarketPulse AI is built with a modern, scalable tech stack:

- **Frontend**: Next.js 14 (App Router), React, Tailwind CSS, Lightweight Charts.
- **Backend**: FastAPI (Python), SQLAlchemy, Pydantic, WebSockets.
- **Database**: PostgreSQL (Relational Data), Redis (Caching & Rate Limiting).
- **Data Providers**: Alpha Vantage (Stocks), CoinGecko (Crypto), NewsAPI (News).
- **Deployment**: Dockerized infrastructure.

## 🚀 Quick Start

### Prerequisites
- Node.js (v18+)
- Python (3.10+)
- PostgreSQL
- Redis

### Backend Setup
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Set up environment variables
cp .env.example .env

# Run database migrations
alembic upgrade head

# Start the server
uvicorn app.main:app --reload --port 8000
```

### Frontend Setup
```bash
cd frontend
npm install

# Start development server
npm run dev
```
